# Auth0 API Implementation
### Learn how to add user authentication to React and API Rest with Nodejs

## Auth0 allows you:
1. Add authentication with different [services](https://auth0.com/docs/connections)
2. Integrate traditional databases (username and password)
3. Support for getting signed [JSON Web Tokens](https://auth0.com/docs/security/tokens/json-web-tokens)
4. Get data from other data sources and add it to the user profile

## Requeriments
1. Get a free [Auth0](https://auth0.com/) Account
2. You have to make sure you have installed:
   - NodeJs >10
   - Terminal
   - Editor
3. Basic understanding of how a website works
